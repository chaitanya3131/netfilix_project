# 🎬 Movie Dataset Exploratory Data Analysis

## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a movie dataset using Python. The analysis was performed to identify trends related to movie popularity, genre distribution, audience voting behavior, and release year patterns.

The project demonstrates data preprocessing, visualization, grouping, aggregation, and insight extraction using popular Python data science libraries.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 Analysis Performed

* Data Cleaning and Preprocessing
* Handling Missing Values
* Genre-wise Analysis
* Popularity Analysis
* Vote Count Analysis
* Release Year Distribution
* Top 10 Popular Movies Analysis
* Data Visualization using Histograms and Barplots

---

## 🔍 Key Insights

* Drama, Action, and Adventure were among the most highly voted genres.
* “Spider-Man: No Way Home” had the highest popularity score in the dataset.
* Most movies in the dataset were released after the year 2000.
* Modern movies dominate the dataset compared to older cinema releases.

---

## 📈 Visualizations Used

* Barplot

These visualizations helped in understanding movie trends and audience preferences.
